  • IO Molecule

    IO Molecule is a simple breakout board for exploring interaction & experience design. It's basically a breakout board for the Adafruit Feather platform with JST-SH connectors and some onboard components.

    • Setup

      Download the Arduino IDE
      Follow the steps in this tutorial to install the Adafruit Boards in the IDE
      Select the Adafruit Bluefruit Micro board from Tools > Board
      Select the correct serial port from Tools > Port
      Open the Blink Example from Files > Examples > 01.Basics > Blink
      Press the 'upload' icon on the top left and see if it works.
      Now download or copy this template project into a sketch and get going!
      If you don't have the Neopixel Library, go to Sketch > Include Library > Manage Libraries and install the Adafruit NeoPixel Library.

    • Pinout

      You can read which Arduino pins belong to which connector on the template file.
      For more details, check out the (somewhat messy) schematic.

    • Inputs

      There are 3 different kind of connectors with 2, 3 & 4 wires each. The black wire is always the ground connection and index 0.
      Button: solder the two wires diagonally.
      Potentiometer: 0 - Left, 1 - Middle, 2 - Right

  • Hyper Island - Experience Prototyping (Nov 2018)

    A two-day workshop for the Hyper Island Digital Media program that explores the physical digital continuum.

    • Brief 1: Create a signal that travels from digital to physical

      Focus on creating something that has very low friction between the digital input and the physical output.

    • Brief 2: Create a signal that travels from physical to digital.

      Make an input device. Explore the relationship between interaction with the input device and the action it triggers on the computer. Explore the affordances for your input. Think about these questions: What do you think you could do with it? What gestures do you do intuitively? What cultural references come to mind?

      To get started, go to Sketch > Include Library > Manage Libraries and install the 'HID-Project' Library.
      Make a new sketch and copy this HID molecule template code into it.
      For more info about HID commands, you can check the HID-project documentation.
      You will probably have reselect the port after every time you upload since the arduino will reboot as an HID device.

      Overall, we recommend to test your inputs without the HID functionality first, as it can be quite tricky to fix bugs :)

    • Brief 3: Create a game that spans both digital and physical.

      You decide what components are physical and which ones are digital. Try to think outside the normal 'sitting still and using your hands' type of games.
